import hashlib
import requests

# Function to check if a password has been leaked
def check_password_leak(password):
    # Hash the password using SHA-1
    sha1_password = hashlib.sha1(password.encode('utf-8')).hexdigest().upper()

    # The first 5 characters of the hash (for k-anonymity)
    prefix = sha1_password[:5]

    # The rest of the hash
    suffix = sha1_password[5:]

    # Query the Have I Been Pwned API with the first 5 characters
    url = f'https://api.pwnedpasswords.com/range/{prefix}'
    response = requests.get(url)

    # Check if the request was successful
    if response.status_code == 200:
        # Check if the suffix is in the response
        hashes = (line.split(':') for line in response.text.splitlines())
        for h, count in hashes:
            if h == suffix:
                return True, count
        return False, 0
    else:
        raise Exception(f"Failed to connect to API: {response.status_code}")

# Function to check multiple passwords
def check_multiple_passwords(passwords):
    results = {}
    for password in passwords:
        leaked, count = check_password_leak(password)
        results[password] = (leaked, count)
    return results

# Example usage
if __name__ == "__main__":
    # You can hardcode passwords here or get them from user input
    passwords = ["password1", "password2", "password3"]

    results = check_multiple_passwords(passwords)

    for password, (leaked, count) in results.items():
        if leaked:
            print(f"Password '{password}' has been leaked {count} times.")
        else:
            print(f"Password '{password}' is safe (not found in known breaches).")

Password Leak Checker Tool

This Python script provides a tool that helps users check if their passwords have been compromised in known data breaches by leveraging the Have I Been Pwned (HIBP) API.

Usage:

The check_password_leak function takes a single argument:

  1. password: The plaintext password you want to check.

To use the tool, follow these steps:

  1. Prepare Your Environment: Ensure you have Python 3 installed on your system. Install the requests library by running: 
    pip install requests
  2. Create a List of Passwords: You can check either a single password or multiple passwords at once. Prepare a list of passwords that you want to verify against known breaches.
  3. Run the Script: Open your Python environment, copy and paste the script into your Python file, replace the placeholder passwords with your actual passwords, and run the script.
  4. Understanding the Output: For each password checked, the tool will inform you whether the password has been found in any known data breaches. If a password has been compromised, the script will also tell you how many times it has appeared in breaches.
  5. Take Action:
    • If a Password is Compromised: Immediately change the password on all accounts where it is used. Make sure to create a strong, unique password for each account.
    • If No Breaches are Found: While your password may not be compromised, it’s still a good idea to regularly update your passwords and ensure they are strong.

Important Warnings:

  • Security of Your Passwords: Be cautious with your passwords. Avoid storing or sharing them insecurely. This tool does not store or transmit your actual passwords, but you should still handle them with care.
  • Avoid Reusing Passwords: Even if a password is found to be safe, reusing passwords across multiple sites can still be a security risk. Always use unique passwords for different accounts.
  • Use a Password Manager: Consider using a password manager to securely store and manage your passwords. Password managers can generate strong, unique passwords for each of your accounts.

Additional Tips:

  • Running the Script on Multiple Systems: You can use this tool on any system that supports Python. Simply ensure you have the required environment set up, and the script will work the same across different platforms.
  • Integrate with Your Security Routine: Regularly check your passwords using this tool as part of your overall security routine. This will help you stay ahead of potential breaches and ensure your online accounts remain secure.
  • Stay Informed: Keep up-to-date with news about data breaches. Knowing when large breaches occur can help you time your checks and take action quickly.

This Password Leak Checker Tool is an easy-to-use solution for ensuring your passwords haven’t been compromised in known data breaches. By following the instructions above, you can proactively protect your accounts and digital assets. Remember to handle your passwords responsibly and take immediate action if any are found to be compromised.
